# Финальная полировка — найденные проблемы и QA checklist

## 1. Найденные проблемы и исправления

| # | Проблема | Приоритет | Статус |
|---|----------|-----------|--------|
| 1 | Отрицательная «экономия» в stats-widgets (когда файл увеличивается при конвертации) | HIGH | ✅ Исправлено: показываем только положительную экономию, 4-й виджет переименован в «Обработано» |
| 2 | formatSizeDiff всегда зелёный (даже при увеличении размера) | HIGH | ✅ Исправлено: добавлена sizeDiffColor() — зелёный при уменьшении, amber при увеличении |
| 3 | operationType показывается как raw string в профиле (image.convert вместо «Конвертация») | MEDIUM | ✅ Исправлено: добавлен formatOp() с маппингом |
| 4 | Слишком много пустого пространства на главной (dropzone p-12, заголовок text-4xl) | MEDIUM | ✅ Исправлено: уменьшен padding до p-8, заголовок до text-2xl/3xl, max-w-2xl |
| 5 | Нет визуальных подсказок о поддерживаемых форматах на главной | LOW | ✅ Исправлено: добавлены теги форматов под dropzone |
| 6 | sizeDiffColor в history-table тоже всегда зелёный | HIGH | ✅ Исправлено |

## 2. QA Checklist перед релизом

### Главная (workspace)
- [ ] Dropzone принимает файл drag-and-drop
- [ ] Dropzone принимает файл через клик
- [ ] После загрузки — toolbar с burger menu
- [ ] Burger menu показывает только совместимые действия
- [ ] Для JPG: убрать фон, resize, compress, rotate, crop, remove-exif, конвертация в PNG/WEBP/AVIF/PDF
- [ ] Для PDF: сжать, разделить, повернуть, удалить метаданные, извлечь страницы, в изображения
- [ ] Для DOCX: в PDF, в TXT
- [ ] Выбор действия → правая панель с параметрами
- [ ] Resize: drag handles работают, поля W/H работают, пресеты работают
- [ ] Remove BG: preview на checkerboard, кисть erase/restore работает
- [ ] Compress: slider quality работает
- [ ] Кнопка «Обработать» → processing → done
- [ ] Скачивание работает (fetch + blob)
- [ ] «Другое действие» → возврат к burger menu
- [ ] «Новый файл» → сброс в idle

### Auth
- [ ] Регистрация: валидация email, пароль 8+ символов, confirmPassword
- [ ] Вход: email + пароль
- [ ] Выход: кнопка в header
- [ ] Protected routes: /profile redirect на /login
- [ ] Refresh token: автоматический при 401

- [ ] Последние операции: правильные названия операций
- [ ] Size diff: зелёный при уменьшении, amber при увеличении
