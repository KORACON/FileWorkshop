## 2. Что ОТЛОЖИТЬ

| Формат/операция | Почему |
|-----------------|--------|
| SVG → редактирование | SVG — векторный, нужен отдельный editor |
| ICO → генерация favicon-пакета | Отдельная фича, не конвертация |
| GIF → анимация | sharp работает только с первым кадром |
| AVIF compress/crop/remove-bg | sharp поддержка AVIF неполная |
| HEIC resize/compress | Зависит от libheif, нестабильно |
| DOCX → HTML | LibreOffice делает грязный HTML |
| Медиа (MP4, MP3) | Тяжёлая нагрузка, отдельный модуль |

## 3. Ограничения и риски

| Риск | Митигация |
|------|-----------|
| SVG через ImageMagick может быть уязвим (XXE) | Только конвертация, не парсинг. Sanitize input |
| HEIC fallback на ImageMagick медленный | Таймаут 120с, предупреждение в UI |
| GIF теряет анимацию | Предупреждение: "Будет конвертирован первый кадр" |
| Большие TIFF (>100MB) | Лимит размера файла |
| ICO через ImageMagick | Только конвертация в PNG/JPG |
